FAQ

Launch notes.

Everything here is made to work as a standalone website that opens the game file.

What file should this website be saved as?

Save this landing page as index.html. Save the game page as skyforgeL1.html in the same folder.

What happens when the Try Game button is clicked?

A classic loading overlay appears, fills the progress bar, then redirects the browser to skyforgeL1.html.

Is it mobile friendly?

Yes. It includes responsive sections, a mobile menu, full-width CTA buttons, and a sticky play bar on smaller screens.

Does this use external assets?

No. The art, layout, animations, preview mockups, and effects are built with HTML, CSS, and JavaScript only.
