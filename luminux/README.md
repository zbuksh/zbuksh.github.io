# Luminux OS Concept

A modern Linux-inspired web OS interface with:

- Login screen
- Glassmorphism desktop
- File Explorer
- Browser shell
- Bash-style terminal
- Settings
- System Monitor
- Notes
- Dock, app launcher and draggable windows
