# NovaDesk Web OS

A modern browser-based Web OS prototype built with plain HTML, CSS and JavaScript.

## Features
- Login/lock screen
- Cyberpunk Linux/KDE-style desktop
- Top system bar with tray, clock, notifications and quick settings
- Dock with modern Font Awesome icons
- App launcher and global search
- Movable, resizable, minimizable and maximizable windows
- File Manager with virtual folders
- Terminal with built-in commands and virtual file system integration
- Settings, Task Manager, Calculator and app placeholders
- Desktop/context menus and keyboard shortcuts

## Run
Open `index.html` in a browser. Internet is required for Font Awesome CDN icons. If you need fully offline icons, download Font Awesome Free and update the CSS link.

## Shortcuts
- Ctrl + Alt + T: Open Terminal
- Ctrl + Space: Global Search
- F11: Browser fullscreen
- Right-click desktop: Context menu
